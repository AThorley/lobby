FA Forever - Lobby
------------------

This is the source code for the FA Forever Lobby.

Pre-requisites are :

- Python 2.6/2.7 32 bits
- PyQt4 with QT 4.7.2
- Python Win32Api
- Py2Exe
- Numpy/Scipy
- Python matplotlib
- FreeImagePy
- bsdiff
- python-dateutil
- six

If you want to contribute back to the project, please make a fork and create
pull-requests of your changes.
